# PVP Tierlist
PVP Tierlist Discord bot.

## Installation
Github:
```
git clone https://github.com/I2rys/pvp-tierlist
```

NpmJS:
```
npm i discord.js@13.16.0
```

## Setup
Open **index.js** and put your Discord bot token in **PvPTL > token**.

## Usage
```
node index.js
```
